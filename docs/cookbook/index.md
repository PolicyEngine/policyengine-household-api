# Cookbook

End-to-end recipes for common partner workflows. Each recipe is a self-contained question with the request and response that answers it. Numbers come from policyengine-us 1.634.8 with 2026 parameters.

## Recipes

- [What's my client's SNAP for 2026?](snap-annual.md) — the simplest useful request, with annual income and an annual benefit
- [When does my client lose SNAP this year?](eligibility-cliff.md) — find the month a benefit cliff fires using month-keyed inputs and outputs

## Suggesting a recipe

If your partner workflow isn't covered here, open an issue at [github.com/PolicyEngine/policyengine-household-api/issues](https://github.com/PolicyEngine/policyengine-household-api/issues) describing the question you want to answer. Recipes are added as patterns repeat across partners.
